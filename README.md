# shree.md

A simple skill to invoke the blessings of the divine by uttering the word "Shree" (श्री) before starting any work. Use this skill when you are beginning a new project or task to seek divine blessings for success and prosperity.


## How to use

Add this skill to your coding agent by adding it to agents skills.

- [VS Code](https://code.visualstudio.com/docs/copilot/customization/agent-skills)
- [Claude Code](https://code.claude.com/docs/en/skills)
- [OpenCode](https://opencode.ai/docs/skills/)
- [Cursor](https://cursor.com/docs/skills)
